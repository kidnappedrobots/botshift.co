Design System

Colors
#0a0a0a bg primary
#111111 bg elevated
#1a1a1a borders
#ff6b35 accent (orange)
#f5f5f5 text primary
#777 text secondary
#444 text muted
Typography
Space Grotesk
Headlines — geometric, techy, distinctive
Geist / DM Sans
Body text — clean, modern, readable
JetBrains Mono
Labels, system text, code references
Principles
→ No glow, no gradients, no blur
→ 1px borders, not card shadows
→ Left-aligned by default
→ Monospace labels for structure
→ Varied section spacing
→ Dot-grid subtle texture
→ Orange used sparingly
→ Border-radius: 4px max
