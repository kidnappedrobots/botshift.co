4. Other visual density techniques used throughout

Dot grid backgrounds
Subtle radial-gradient dot pattern behind key sections. Creates depth without imagery.
Monospace system labels
"// section-name" prefixes, numbered sequences, status indicators. Terminal personality.
1px grid dividers
Instead of cards with borders, use 1px lines to create structure. Feels architectural.
Purposeful SVG icons
Custom line-drawn icons for each agent/concept. Consistent stroke weight, orange accent.
Asymmetric layouts
Not everything centered. Left-aligned text, varied section widths, intentional negative space.
2 real images where they matter
Vision section gets the hero image. Agents section gets one hero image. These land harder because the rest is restrained.

The page isn't "without images" — it's full of visual information. Photos are replaced by engineered elements that reinforce the mission-control personality.
